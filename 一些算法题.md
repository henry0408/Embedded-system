
## 1.给定一个二维数组，其每一行从左到右递增排序，从上到下也是递增排序。给定一个数，判断这个数是否在该二维数组中。

Consider the following matrix:
[
  [1,   4,  7, 11, 15],
  [2,   5,  8, 12, 19],
  [3,   6,  9, 16, 22],
  [10, 13, 14, 17, 24],
  [18, 21, 23, 26, 30]
]

Given target = 14, return true.
Given target = 20, return false.

```
#include <stdio.h>

int main()
{
    int arr[5][5] = 
    {
        {1,   4,  7, 11, 15},
        {2,   5,  8, 12, 19},
        {3,   6,  9, 16, 22},
        {10, 13, 14, 17, 24},
        {18, 21, 23, 26, 30}
    };

    int num = 14;

    int col = 4;
    int row = 0;

    while (row>=0 && row<=4 && col >=0 && col <=4)
    {
        printf("%d %d\n",row,col);
        //printf("%d\n",arr[row][col]);
        
        if(arr[row][col] > num)
        {
            --col;
        }
        if(arr[row][col] < num)
        {
            ++row;
        }
        if(arr[row][col] == num)
        {
            printf("%d\n",num);
            break;
        }
    }
}
```
